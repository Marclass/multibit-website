Keyboard shortcuts

MultiBit Classic has keyboard shortcuts that you can use to save time moving around the screen. They are:

  • Shift-ArrowUp. When the focus is in the Wallets list, select the wallet above in the list of wallets. This moves the selected wallet up one.
  • Shift-ArrowDown. When the focus is in the Wallets list, select the wallet below in the list of wallets. This moves the selected wallet down one.
