BlockStoreException

You see a BlockStoreException very occasionally when MultiBit Classic starts. It prevents you synchronising with the Bitcoin network.

You can fix this problem by doing the following:

  1. Find where MultiBit Classic is storing your user data - see the help topic Troubleshooting

  2. Shut down MultiBit Classic

  3. Go to your MultiBit Classic user data directory - worked out in step 1 - and delete the file multibit.spvchain

  4. Start up MultiBit Classic again

It will then regenerate this file and start syncing. After it has synced you should see all your bitcoin in your wallet..
