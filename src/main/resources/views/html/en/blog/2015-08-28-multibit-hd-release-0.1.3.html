MultiBit HD Release 0.1.3 is now available

We are pleased to announce the next MultiBit HD Release (0.1.3) is available for download.

This is a maintenance release.

Summary of fixes

Here is a quick summary of the fixes contained in this release:

  • Greek translation completed
  • Remove Tor support (temporarily)
  • Reduced number of unnecessary error report triggers
  • Reduced BRIT fees so that small amounts are not charged on Empty Wallet
  • Updated Windows/OS X developer certificates
  • Update Bitcoinj library to 0.13.2
  • Various small user interface tidy ups

Greek translation completed

Thanks to the translation efforts of user johnblazakis we now have a complete translation of MultiBit HD into Greek.

Temporarily dropping Tor support

It is with regret that we have had to pull Tor support from the wallet temporarily. This is due to a number of error reports that we received indicating problems out in the field and our own experiences attempting to use Tor on a variety of networks.

Our Tor support comes from the Orchid library fork integrated with the excellent Bitcoinj library and was presented under our Labs section as an experimental feature while we tried it out on a larger user base. Unfortunately there are some problems with it right now that prevent it from working reliably enough to continue. Consequently we have decided to switch it off while we work with the Orchid/Bitcoinj team to resolve the issues so it can be reinstated in a later release.

For more information on the discussion leading to this decision, please see the following GitHub issues:

  • #633 Upgraded to Orchid 1.1.1 (Final decision based on TimeoutException)
  • #634 Orchid 1.0 throws UnsupportedOperationException (fixed in Orchid 1.1.1)
  • #666 Representative Tor failure from the field (with screen shots)

Semi-automatic update

We normally wait 24 hours after a release and then activate the semi-automatic upgrade process. You decide if the upgrade occurs. The upgrade process will not affect your wallet data. Only the application binaries will be upgraded.

Note that MultiBit HD 0.1.2 did not have the upgrade process enabled so you will have to download the installer directly from our download page.

You can find more details, including digital signatures, on the download page.
